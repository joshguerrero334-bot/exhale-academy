Exhale Academy is a subscription-based TMC prep platform for RRT students and repeat test takers.

## Tech Stack
- Next.js + TypeScript (frontend)
- Supabase (authentication + database)
- Stripe (subscriptions)
- Vercel (hosting)

## Core Product Loop
1. User signs up / logs in
2. User subscribes ($29/month)
3. User takes quizzes by TMC category
4. User reviews score + rationales + keyword traps
5. Progress tracking highlights weak areas
