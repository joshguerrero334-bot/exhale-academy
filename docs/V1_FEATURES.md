# Exhale Academy — V1 Features (TMC Only)

## Must-Have
- User authentication (email/password)
- Stripe subscription paywall ($29/month)
- Question bank tagged by TMC outline categories
- Quizzes (10/20/50 question sets)
- Instant scoring + review
- Detailed rationales (“why correct” + “why others are wrong”)
- Progress tracking by category

## Nice-to-Have (Post-V1)
- Timed mode
- Weak-area insights
- Bookmark questions
- Admin question editor UI
